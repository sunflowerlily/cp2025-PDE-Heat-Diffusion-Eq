## 2. 理论背景

### 2.1 含时薛定谔方程

一维含时薛定谔方程的标准形式为：

$$i\hbar \frac{\partial \psi(x,t)}{\partial t} = -\frac{\hbar^2}{2m} \frac{\partial^2 \psi(x,t)}{\partial x^2} + V(x)\psi(x,t)$$

其中：
- $\psi(x,t)$ 是波函数
- $\hbar$ 是约化普朗克常数
- $m$ 是粒子质量
- $V(x)$ 是势能函数

### 2.2 方势垒模型

本实验采用方势垒模型：

$$V(x) = \begin{cases}
0 & x < x_1 \text{ 或 } x > x_2 \\
V_0 & x_1 \leq x \leq x_2
\end{cases}$$

### 2.3 初始波包

高斯波包的初始条件：

$$\psi(x,0) = \left(\frac{1}{2\pi\sigma^2}\right)^{1/4} \exp\left(-\frac{(x-x_0)^2}{4\sigma^2} + ik_0x\right)$$

### 2.4 隧穿系数

透射系数定义为：
$$T = \frac{\text{透射区域的概率}}{ \text{入射区域的概率}}$$

反射系数定义为：
$$R = \frac{\text{反射区域的概率}}{ \text{入射区域的概率}}$$

理论上应满足：$T + R = 1$

## 3. 数值方法

### 3.1 Crank-Nicolson格式

将薛定谔方程离散化，采用Crank-Nicolson隐式格式：

$$\psi_j^{n+1} = \psi_j^n + \frac{i\Delta t}{2\hbar}\left[H\psi^{n+1} + H\psi^n\right]_j$$

其中哈密顿算符的离散形式为：
$$H_j = -\frac{\hbar^2}{2m} \frac{\psi_{j+1} - 2\psi_j + \psi_{j-1}}{(\Delta x)^2} + V_j\psi_j$$

### 3.2 矩阵形式

整理得到线性方程组：
$$A\psi^{n+1} = B\psi^n$$

其中 $A$ 和 $B$ 是三对角矩阵。

### 3.3 边界条件

采用吸收边界条件：$\psi(0,t) = \psi(L,t) = 0$
